# Todo-App

Todo app with flutter.In this version there is no database or an API.
## Installation


```bash
git clone https://github.com/ahmetbekir22/todo-app-flutter.git
```

## Usage

```bash 
flutter pub get
```
```bash 
flutter run 
```
